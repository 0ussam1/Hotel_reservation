# 🏨 Application de gestion des réservations d’hôtel – Streamlit + SQLite

Bienvenue dans ce projet de gestion d’un système de réservation d’hôtel.  
Développé avec **Python**, **SQLite** et **Streamlit**, ce projet permet de gérer des clients, des réservations et des chambres à travers une interface web simple et efficace.

---

## 🔧 Fonctionnalités de l’application

L’interface permet d’effectuer les actions suivantes :

1. 📅 **Consulter les réservations**
2. 👥 **Consulter la liste des clients**
3. 🛏️ **Afficher les chambres disponibles pendant une période donnée**
4. ➕ **Ajouter un nouveau client**
5. 📝 **Ajouter une réservation**

---

## 🧠 Technologies utilisées

- [Python](https://www.python.org/)
- [SQLite](https://www.sqlite.org/index.html)
- [Streamlit](https://streamlit.io/)
- [Pandas](https://pandas.pydata.org/) (pour la gestion des données)
